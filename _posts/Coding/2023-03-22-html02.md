# Layout

## 1. div: division

```html
<!-- HTML -->
<div class="box">
	<p>This is a Box</p> <!--CSS offers inherit from mother to children-->
</div>
```

```css
/* CSS */
.box {
	width: 200px;
	background-color: green;
	margin: 20px 30px; /*blank space, top/bottom left/right*/
	padding: 20px; /*inside blank space*/
	border: 4px dotted yellow;
	border-radius: 5%;
	/*center aligned div*/
	margin-left: auto;
	margin-right:auto;
	font-size: 12px;
	box-shadow: 5px 5px;

}
```

## 2. float

- **Right or Left Alignment**

```html
<!-- HTML -->
<!-- layout.html -->
<div class="container">
	<div class="header">
		header
	</div>
	<div class="sidebar">
		sidebar
	</div>
	<div class="main">
		main
	</div>
	<div class="footer">
		footer
	</div>
</div>
```

```css
/* CSS */
/* imported by layout.html */
.container {
	width: 1000px;
	color: white;
  text-align: center;
}
.header {
	width: 100%;
	height: 60px;
	background: black
}
.sidebar {
	width: 25%;
	height: 500px;
	background: blue;
    float: left;
}
.main {
	width: 75%;
	height: 500px;
	background: coral;
    float: left;
}
.footer {
	width: 100%;
	height: 60px;
	background: black;
    clear: both; /*solve the strange phenomenon caused by the float*/
}
```

## 3. inline-block

- display: block (take a whole row)
- display: inline-block (take own space)
- inline-block annoying to remove spaces between empty space when used (No spaces required to be placed in the same row)
```html
<div class="container">
	<div class="header">
		header
	</div>
	<div class="sidebar">
		sidebar
	</div><div class="main">
		main
	</div>
	<div class="footer">
		footer
	</div>
</div>
```

```css
/* CSS */
/* imported by layout.html */
.container {
	width: 1000px;
	color: white;
  text-align: center;
}
.header {
	width: 100%;
	height: 60px;
	background: black
}
.sidebar {
	width: 25%;
	height: 500px;
	background: blue;
  display: inline-block;
}
.main {
	width: 75%;
	height: 500px;
	background: coral;
  display: inline-block;
}
.footer {
	width: 100%;
	height: 60px;
	background: black;
}
```

## 4. Practice  - Blog List

- Blog List Layout

```html
<!--HTML-->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link href="./index.css" rel="stylesheet" >
</head>
<body>
    <div class="blog_list">
        <div class="blog_content">
            <div class="list_header">
                <img class="header_img" src="./unnamed.gif"> 
                <h4 class="header_name">Brooklyn Bridge</h4>
            </div>
            <hr style="clear: both;"/>
            <div class="list_body">
                <h5 class="body_title">
                    Landmark of NYC
                </h5>
                <p class="body_content">
                    &nbsp;The Brooklyn Bridge is a hybrid cable-stayed/suspension bridge in New York City, 
                    spanning the East River between the boroughs of Manhattan and Brooklyn. Opened on May 24, 1883, 
                    the Brooklyn Bridge was the first fixed crossing of the East River. 
                </p>
            </div>
        </div>
        <img class="list_img" src="https://cdn.pixabay.com/photo/2013/04/16/14/22/brooklyn-bridge-105079_960_720.jpg">
    </div>      
</body>
</html>
```

```css
/*CSS*/
.blog_content {
	width: 75%;
	height: 160px;
	background-color: black;
	color: white;
	float: left;
}
.list_header {
	text-align: center;
	margin-left: 10px;
}
.header_name {
	float: left;
	margin: 8px;
}
.header_img {
	float: left;
	height: 30px;
}
.list_img {
	width: 25%;
	height: 160px;
	float: left;
}
.list_body {
	clear: both;
	margin-top: 10px;
	margin-left: 10px;
}
```
